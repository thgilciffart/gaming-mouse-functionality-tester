# clicks

- click to your hearts content! left click, right click, middle click and backwards and forwards.

- clicks will only register as a double click if the consecutive click is within the threshold set by the ΔT variable. for example if you click once and then again 50ms after which is under the default threshold, this will register as a double click

# scroll

- scroll any direction you want! up down left right

- you may need a trackpad to scroll left and right though

- this is useful as some mice are faulty and register a scroll in the opposite direction. for example for every 100 scrolls down it will register a false scroll up.

# polling rate test

- useful to check if your mouse is running at the correct polling. if you are using windows 11 23h2 (check by running winver in the win+r menu) then you must be selecting the window
