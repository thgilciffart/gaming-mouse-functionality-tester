# Gaming Mouse Functionality Tester
A completely free, open-source offline webapp that is designed to test if your mouse is fully functional and works as advertised

## Features
- Click meter for all common buttons on the mouse (LMB,RMB,MMB,MB4,MB5)
- A scroll wheel test to see if your trackpad or scroll-wheel is fully functional. Some mice scroll up even when scrolling down which can be annoying.
- A double click feature to see if your mouse has any double clicks. The threshold is adjustable.
- A CPS test just for fun
- A polling rate test to see if your mouse's polling is fully up to spec of the advertised speed

## Usage
You can either use the tool [here](https://mouse-functionality-tester.netlify.app/) or use it locally.
1. Clone the repo 
```bash 
git clone https://github.com/thgilciffart/gaming-mouse-functionality-tester.git
```
2. 
```bash 
cd gaming-mouse-functionality-tester
```
3. Open ``index.html``
